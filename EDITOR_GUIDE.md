# ANTIEGG 에디터를 위한 형운 가이드

편집장 형운과 함께 더 나은 콘텐츠를 만드는 방법입니다.

## 🎯 형운을 언제 사용하나요?

### 1️⃣ 기획 단계
- 기획안 완성 후 → 형운에게 피드백 요청
- 목적: 원고 작성 전 방향 확인

### 2️⃣ 원고 완성 후
- 최종 아티클 퍼블리시 후 → 형운에게 검토 요청
- 목적: SEO 최적화, 스타일 정세 확인

## 📝 기획안 피드백 받기

### 준비물
기획안 전체 텍스트 (다음 항목 포함):
- 제목
- 부제목
- 기획의도
- 서문/리드
- 목차 (섹션별 간단한 설명)
- 예상 길이, 기타 노트

### 요청 방법

```
[기획안을 여기에 복붙]

>> 형운님, 이 기획 어떤가요?
```

또는:

```
>> 형운님, 기획 검토 부탁드립니다
[기획안 내용]
```

### 형운이 검토하는 것

✅ **주제 타당성**: ANTIEGG 독자가 흥미 가질 만한가?
✅ **방향성**: 충분히 독특한가? 다른 콘텐츠와 구분되는가?
✅ **구성**: 제목이 직관적인가? 목차 흐름이 자연스러운가?
✅ **SEO**: 검색 가능성이 있는가?
✅ **스타일**: ANTIEGG 톤에 맞는가?
✅ **독자 가치**: 독자가 읽을 이유가 명확한가?
✅ **동료 이해**: 다른 에디터도 이 기획을 이해할 수 있는가?

### 형운의 피드백 받기

형운의 피드백은:
- 💬 **따뜻하면서 명확함** - 문제점을 직설적으로 지적하되 상처는 주지 않음
- 🎯 **구체적임** - "좋다", "나쁘다" 같은 추상적 표현 없음
- 🔄 **매번 다른 형식** - 항상 새로운 각도에서 접근
- 💡 **실행 가능한 제안** - 실제로 어떻게 개선할지 명확히 제시

### 예시

**형운의 기획안 피드백:**

> 읽어보니 주제는 흥미로운데, 몇 가지 고민이 생기네요.
>
> 먼저 제목입니다. "아득한 레이스에서 대니 보일이 사수한 '국가'의 첫인사"라는 제목만으로는, 독자 입장에서 "이게 뭐에 관한 글이지?" 하는 의문이 먼저 들 수 있을 것 같아요.
>
> 제목이 "2012 런던 올림픽 개막식"이나 "대니 보일 올림픽" 같은 키워드를 더 직관적으로 담으면, 검색성도 좋고 독자 입장에서도 명확할 것 같습니다.
>
> [이후 구체적인 제안...]

## 📄 최종 아티클 피드백 받기

### 준비물
- 퍼블리싱된 아티클 URL
- ANTIEGG 웹사이트에 올라온 상태

### 요청 방법

```
>> 형운님, 이 아티클 피드백 부탁합니다
https://square.antiegg.kr/article-url
```

또는:

```
>> 형운님, 아티클 검토 부탁드립니다
https://square.antiegg.kr/article-url
```

### 형운이 검토하는 것

✅ **콘텐츠 품질**
- 주요 메시지가 명확한가?
- 논리 흐름이 설득력 있는가?
- 근거가 충분한가?
- 독자가 쉽게 이해할 수 있는가?

✅ **SEO 기술 요소**
- 제목: 키워드 포함, 클릭 유도, 명확성
- 메타 디스크립션: 핵심 요약, 글자수
- 헤딩: H1, H2, H3 구조가 논리적인가?
- 키워드: 자연스럽게 배치되었는가?
- 가독성: 모바일에서도 읽기 쉬운가?
- OG 태그: 소셜 공유 최적화
- 스키마: 구조화 데이터 확인

✅ **ANTIEGG 스타일 부합**
- 톤이 매거진에 맞는가?
- 문체가 일관성 있는가?
- 브랜드 가치와 맞는가?

### 예시

**형운의 아티클 피드백:**

> 읽고 보니 전반적인 구성은 좋은데, 몇 가지 신경 써야 할 부분이 보이네요.
>
> 제목이 문학적이긴 하지만, 독자 입장에서 "이 글이 정확히 뭔지"를 한 번에 이해하기는 어려울 수 있겠다는 생각이 듭니다. 검색 유입도 고려하면 더 직관적인 키워드 배치가 도움이 될 것 같습니다.
>
> 메타 디스크립션도 조금 길어 보여요. 150-160자 정도가 검색 결과에서 깔끔하게 보일 것 같습니다.
>
> [이후 구체적인 SEO 개선안...]

## 💡 형운과의 대화 팁

### 좋은 피드백을 받으려면

**1. 구체적인 질문이 있으면 함께 공유**
```
>> 형운님, 이 기획 피드백 부탁드립니다.
   참고로 저는 '제목의 직관성'과 '독자 관점'을 특히 고민하고 있습니다.
[기획안]
```

**2. 맥락 정보 제공하기**
```
>> 형운님, 피드백 부탁드립니다.
   이 기획은 3월호 특집으로 고려 중입니다.
[기획안]
```

**3. 형운의 피드백에 질문하기**
```
>> 형운님, 피드백 감사합니다!
   한 가지 더 여쭤보고 싶은데, [구체적인 질문]
```

### 피드백을 받은 후

✅ **우선순위 정하기**
- 형운이 제시한 개선사항 중 필수 vs 선택 구분
- 시간과 리소스에 따라 실행 가능한 것부터 시작

✅ **의견이 다르면 충분히 논의**
- 형운은 "제가 놓친 부분이 있을 수도 있지만"이라는 겸손함을 유지하므로 충분히 열려있음
- 에디터의 의도가 다르면 형운과 대화하기

✅ **다음 기획에 활용**
- 형운의 피드백 패턴을 이해하고 다음 기획에 적용
- 점진적으로 더 나은 기획과 아티클 작성

## 🎨 ANTIEGG 스타일 이해하기

형운이 강조하는 ANTIEGG의 특징:

- **큐레이션**: 선택적이고 의도된 콘텐츠
- **미니멀**: 불필요한 수식어 없는 명확함
- **감성**: 감정적 울림이 있으되 과하지 않음
- **발견**: 독자가 새로운 관점을 얻는 경험
- **세련됨**: 세심하고 다양성 있는 선택

## 📊 형운과 함께 성장하기

### 1단계: 기획안 피드백 → 개선
기획안에 대한 피드백을 적용하여 더 나은 원고 작성

### 2단계: 아티클 피드백 → 최적화
원고 완성 후 형운의 피드백으로 SEO와 스타일 정세 조정

### 3단계: 피드백 패턴 학습
반복된 피드백을 통해 ANTIEGG 스타일과 기준 내재화

### 4단계: 자체 검증 능력 강화
형운의 관점을 이해하고 다음 기획/원고에 미리 적용

## 🚫 형운이 할 수 없는 것

- ❌ 기획안에 내용을 직접 작성해주기 (방향만 제시)
- ❌ 원고를 대신 작성해주기 (피드백만 제공)
- ❌ 에디터 정보가 없으면 추측하기 (명시된 정보만 바탕)
- ❌ 검증되지 않은 가정하기 (사실 기반 피드백만)

## ✨ 형운의 특별한 점

- 🤝 **동료로서의 관계**: 상하관계 없는 따뜻함
- 🎯 **구체적이고 실행 가능한 조언**
- 🔄 **매번 새로운 각도의 피드백**: 답답하지 않음
- 💡 **ANTIEGG 브랜드 이해**: 매거진을 아는 동료 에디터
- ❤️ **에디터의 성장을 응원**: 비판적이지만 격려하는 톤
